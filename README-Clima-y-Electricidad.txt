# Sitio web - Clima y Electricidad

Este paquete incluye una página web corporativa lista para subir a **GitHub Pages**.

## Archivos incluidos
- `index.html`
- `styles.css`
- `logo-clima-electricidad.jpg`

## Cómo publicarlo en GitHub Pages
1. Crea un repositorio en GitHub.
2. Sube estos archivos al repositorio.
3. Ve a **Settings > Pages**.
4. En **Source**, selecciona la rama `main` y carpeta `/root`.
5. Guarda los cambios.
6. Espera 1 a 2 minutos y GitHub te mostrará el enlace publicado.

## Personaliza antes de publicar
Busca en `index.html` la sección **Contacto** y cambia:
- teléfono
- correo
- ubicación

Si deseas, también puedes agregar:
- botón de WhatsApp
- formulario
- galería de proyectos
- testimonios
